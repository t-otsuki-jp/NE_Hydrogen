# Hydrogen supply model "NE_Hydrogen"
The New Earth_Hydrogen model (NE_Hydrogen) is a two-region, single-year, temporally detailed hydrogen supply model developed by Takashi OTSUKI (Associate Professor, Yokohama National University), Tatsuya HAGITA (Senior Researcher, The Institute of Energy Economics, Japan (IEEJ)), and Yoshiaki SHIBATA (Senior Research Director, IEEJ). This model is written in Xpress-Mosel.

## File description
- NE_Hydrogen.mos | Model file
- NE_Hydrogen_Input.xlsx | Inputdata file
- NE_Hydrogen_Output.xlsx | Annual and hourly results
- NE_Hydrogen_Output2.xlsx | Seaborne transportation capacity and operation results
